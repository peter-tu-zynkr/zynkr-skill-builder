# 篩選關鍵字的 SOP

對應流程圖綠色節點「篩選關鍵字的 SOP」。分群前先篩。

## 剔除原則

- 與品牌服務無關的字（純資訊、無商業關聯）。
- 品牌明確不打的市場 / 受眾。
- 重複或近義過度的字（合併為主字 + 變體）。
- 搜尋量趨近 0 且非策略性長尾。

## 保留優先

- 對應到人物誌真實痛點的字。
- BOFU 決策意圖字（顧問、ROI、比較、推薦）—漏斗目標。
- 可用 Zynkr 第一手證據差異化的字。
- AEO 友善的問句型整句。

## 分群前提醒

- 先意圖、後類別。
- 一字可暫時掛在多群，分版本時再決定主歸屬。
- 標記語言（zh-TW / 旗艦 EN）。
