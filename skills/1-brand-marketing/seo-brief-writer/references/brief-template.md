# SEO 撰寫 Brief 範本

對應流程圖綠色節點「Brief 格式」。每篇文章一份。

```
# SEO Brief：<文章工作標題>

## 基本
- 目標關鍵字（主）：
- 輔助關鍵字：
- 搜尋意圖：資訊 / 商業調查 / 交易決策
- 漏斗階段：TOFU / MOFU / BOFU
- 內容支柱：<五大支柱之一>
- 語言：zh-TW（旗艦 EN：是 / 否）

## 讀者
- 目標人物誌：<來自 Persona 包>
- 讀者目前認知 / 痛點：
- 讀完後能做到什麼（desired outcome）：

## 角度與勝出
- 價值主張（一句）：
- 勝出關鍵（2–3 點）：
- 對比天花板文章的差異：

## 內容
- 必涵蓋重點（bullet）：
- 第一手素材 / 案例 / 數據：（不足標 [需 Peter 補]）
- 必答 FAQ（給 AEO）：3–5 題

## 轉換
- 主要 CTA：<B2B 優先：預約診斷 / discovery call；或 lead magnet>
- 內部連結對象：<支柱頁 / 相關文章 / /consult>

## AEO 要求
- 開頭即答（answer-first）一句：
- 需要的結構化區塊：FAQ / 比較表 / 步驟 / schema
- 一個可被引用的「命名框架」：
```

## 使用

欄位填不滿就標記，交棒前經人工「驗證並補充」。Brief 是大綱與寫作的唯一真實來源。
