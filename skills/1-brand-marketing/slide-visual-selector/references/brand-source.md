# 品牌視覺來源（設定，非內建）

> **本技能不內建任何品牌內容。** 品牌指南屬於內部資訊，不放進這個公開技能庫。
> 本檔只做兩件事：(1) 告訴技能**去哪裡載入**你的品牌視覺規範；(2) 給一份**通用 schema**，說明要從品牌指南萃取哪些東西、怎麼套進「設計備註」。
> 實際色值、字體、影像規範一律在**執行時**從你設定的來源載入，不寫死在這裡。

---

## 1 · 設定品牌來源（編輯這裡）

```
BRAND_GUIDE = <你的品牌指南位置>
```

- **本機檔案**：用 Read 讀取你的品牌指南（Markdown / PDF / token 檔）。
- **Google Drive**：用 `google-workspace` MCP 依名稱 search 你的品牌指南，再讀內容。
- **Zynkr 內部預設**：品牌指南檔名 `Zynkr-Brand-Guide.md`（位於品牌資料夾），含色彩系統、字體級距、影像法則、logo 資產；以本機 Read 或 Drive 搜尋載入。

**找不到 / 未設定時** → 走「中性 fallback」（見 §4），並明確告訴使用者「目前用中性視覺預設，未套用品牌」。

---

## 2 · 要從品牌指南萃取什麼（通用 schema）

載入後，把下列角色對應到品牌指南裡的實際值，再寫進每頁「設計備註」。這些是**通用設計角色**，任何品牌都適用：

| 角色 | 說明 | 套用在 |
|---|---|---|
| **預設表面色 (surface)** | 頁面預設底色 | `slide.background` |
| **深對比色 (dark contrast)** | 全幅深底（封面/章節/結尾） | 深底頁背景；文字翻成表面色 |
| **強調色 (highlight)** | 思考/關鍵字強調 | 標題關鍵字，**每標題 ≤ 1** |
| **決策色 (accent / decision)** | 唯一的決定、CTA | **每頁最多 1 次** |
| **文字色 / 註腳色** | 內文與次要資訊 | addText 顏色 |
| **字體角色** | 顯示字 / 內文字 / 數字編號字 / 中文字 | pptx `fontFace` |
| **字級級距** | 顯示 / 標題 / 內文 / 註腳 / 編號 的 pt | pptx `fontSize` |
| **影像法則** | 偏好結構性圖解 vs 裝飾性圖片；母題；icon 規格 | image-led / concept / divider 頁 |
| **logo 資產** | 深底用淺色 logo、淺底用深色 logo | 封面/結尾 `addImage` |

---

## 3 · 通用設計原則（非品牌專屬，可直接守）

即使沒有品牌指南，這些原則一律成立：

- **一個決策色、每頁最多 1 次**：只標真正的「決定 / CTA / 關鍵數字」。整頁都在強調 = 沒有重點。
- **強調色節制**：關鍵字才上強調色，多了就稀釋。
- **不裝飾，要圖解**：能用結構性圖解（流程 / 層級 / 矩陣 / 比較面板）說清楚關係，就別塞氣氛用的 stock 圖。
- **深—淺—深三明治**：封面深底 → 內頁淺底 → 結尾深底，收尾呼應開場。

---

## 4 · 中性 fallback（未設定品牌時）

- 表面色：白／淺灰；深對比：深灰近黑；決策色：單一鮮明色（每頁 1 次）；文字：深灰。
- 字體：sans-serif 一套（顯示加粗、內文常規）；中文用系統黑體。
- 影像：仍守「不裝飾，要圖解」。
- 並提示使用者：可在 §1 設定品牌來源以套用實際品牌。

---

## 5 · pptxgenjs 落地（通用機制，非品牌內容）

- 顏色值**去掉 `#`**：`fill: { color: "RRGGBB" }`、`color: "RRGGBB"`。
- 預設底色：`slide.background = { color: "<surface>" }`。
- 16:9 寬版畫布 ≈ `13.33" × 7.5"`（`LAYOUT_WIDE`），預留 `0.5"` 邊界。
- 字體在算繪機若不可用，pptx 技能會 fallback；本棒只需在設計備註指定「字體角色」。
