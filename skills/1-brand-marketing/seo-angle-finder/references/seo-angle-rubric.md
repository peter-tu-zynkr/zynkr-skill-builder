# SEO 切角評估 Rubric

對應流程圖「哪裡可以加強」的思考。為每個候選切角評估三個維度。

## 三維度

1. **競品缺口**：把競品（已知對手網址）涵蓋的點列出 → 找出 missing / 講不清 / 過時的地方。
2. **Zynkr 差異化證據**：這個切角能不能用第一手證據撐起來？
   - 直播實作（CRM / CMS / 會計 / skills pipeline）
   - 顧問方法論（SIPOC、FE/BE/DB swimlane、Digital Readiness Scorecard、RICE）
   - 真實數據 / 案例（去識別化）
   - proof-over-promise：能「show the build」勝過「講道理」。
3. **AEO 搶引用潛力**：這題能否用「答案寫在最前面 + 結構化」搶下 AI 引擎引用？

## 評分

每個切角給三維度各 1–3 分；總分高者優先。**至少要有一項 Zynkr 差異化證據**，否則退回（避免變成沒有第一手依據的泛泛內容，違反品牌定位）。

## 輸出每個切角

- 切角標題
- 對應買家問題（來自 Questions 包）
- 競品缺口一句話
- Zynkr 差異化證據一句話
- AEO 潛力（高／中／低）
