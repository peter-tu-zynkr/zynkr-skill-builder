# Skills Source

This is the canonical source for all SKILL.md files. Each subfolder maps to a taxonomy category.

Empty category folders are valid placeholders — they exist to reserve the category slot and will be populated as skills are added.

## Folder naming

```
[N]-[category-slug]/
  [skill-slug]/
    SKILL.md
    agents/          ← subagent .md files (orchestrator skills only)
```
